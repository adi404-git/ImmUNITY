ImmUNITY is a 2D action-survival game set inside the human bloodstream, where the player fights invading pathogens using movement, timing, and biological combat mechanics.
The project was developed as part of a game development task by WinterOfGameDev (WoG), CGS, with a strong focus on player progression, skill systems, and gameplay feedback.
